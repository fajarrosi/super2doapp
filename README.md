## How to Deploy

```bash
# create Database
Create Database with db name => super2doapp

# copy File .env.example 
cp .env.example .env

# generate key laravel
php artisan key:generate

# migrate the database
php artisan migrate

# composer dump-autoload
composer dump-autoload

# serve with hot reload 
php artisan serve

```
